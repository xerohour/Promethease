# Customer Update - 2026-03-03

We shipped a usability and visual refresh for the 2026 report experience.

## What's New

- 2026 UI2 reports now include a right-side **Search + Filters** panel.
- You can now filter by text, repute, and minimum magnitude in one place.
- A built-in help walkthrough is now available directly in the 2026 UI2 report pages.

## What's Better

- Report navigation pages now share a cleaner, consistent visual style.
- The 2026 UI2 filter behavior now feels closer to the classic Promethease interaction model.

## Bug Fixes

- Resolved a filter panel rendering issue caused by Intro.js runtime classes being present in static page markup.
